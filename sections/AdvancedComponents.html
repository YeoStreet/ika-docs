Optional parameters on creating a component

When using the createIkaComponent() method, you will always need to pass the component's name as a string in the first parameter. Optionally, you can pass in a second parameter as an object with the following keys:
  • templateId: Gives a template element ID that is not the default (same as component name).
  • construct: Passes in a function to be executed in the constructor. It will have this bound to the component class.
  • eventBinds: Specifies what events the component will listen to and capture, as an array of strings. All Ika components will listen to and capture click events, as well as Ika internal events that bubble as CustomEvent.
