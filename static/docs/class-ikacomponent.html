class IkaComponent

The IkaComponent class extends HTMLElement interface, and therefore has the same properties and methods outlined here. Additional options are outlined below.

Instance properties

None, except those inherited from HTMLElement.

Instance methods

IkaComponent.getState()

Gets the state of a component instance specified by a state key.

Syntax

getState(stateKey)

Parameters

stateKey: string

The key identifying the state value to be retrieved.

Return value

The current value of the state as identified by the state key.

IkaComponent.setState()

Sets the state of a component instance specified by a state key.

Syntax

setState(stateKey, value)

Parameters

stateKey: string

The key identifying the state value to be set.

value

The value to be set on the state key.

Return value

None.

IkaComponent.linkState()

Invokes a callback function passed in as an argument whenever a state's value is changed.

Syntax

linkState(stateKey, cb)
linkState(stateKey, cb, linkedStateKey)

Parameters

stateKey: string

The state key to watch value changes on.

cb: (this: IkaComponent, v) => any

A callback function when the state's value is changed. The function passed in will receive one argument that is the new state value, and have its this bound to the component instance.

linkedStateKey: string

The return value from the cb function will be assigned to the state key matching linkedStateKey. This is equivalent to using setState() for the linked state key.

Return value

As defined by the function being invoked.

IkaComponent.invokeFunction()

Syntax

invokeFunction(methodName)
invokeFunction(methodName, ...args)

Parameters

methodName: string

The key referencing a functions supplied in the bundle.m KV object, when defining the component with defineIkaComponent().

args

Additional parameters to be passed to the referenced function.

Return value

As defined by the function being invoked.
