A web components framework

Read docs

Standards driven

Ika aims to reduce the JS written for web front-ends, using native Web APIs wherever possible.

Ergonomics first

Ika prioritises developer ergonomics, and aims to be as unopinionated and useful as possible to get your project up and running.

Zero dependency

Reducing complexity and improving compatibility, Ika has no package dependencies and comes at roughly ~15kb gzipped.

Get started

Add the component template to your HTML:

<ika:templates>
   <template id="hello-world">
       Hello, world!
   </template>
</ika:templates>

Then, declare the component through JS:

<script type="module">
   import { defineIkaComponent } from 'https://unpkg.com/ikajs';
   defineIkaComponent('hello-world')
</script>

Done! Use your component as <hello-world> anywhere on your page.
